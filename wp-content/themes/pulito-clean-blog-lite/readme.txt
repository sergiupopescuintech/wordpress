== Description ==

Pulito Clean Blog Lite is a beautiful WordPress blog theme for blogs website. Theme is lightweight, fast and optimized for all mobile phones.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Upload pulito-clean-blog-lite.zip file.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel.
